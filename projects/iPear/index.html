iPear® XS™

iPear XS is a satirical smartphone simulator written in JPHP (JVM) for Windows 10/11 after an inside joke about Apple® iPhone®. I have never had an iPhone®, so the actual design is based on Android smartphones. The phone contains several (ultimately useless) "apps", such as a weather app (embedded yaWeather), a browser, an audio player, a calculator, a translator, and several more. You can also change several "settings" (including Wi-Fi connections and the wallpaper) and even customize your phone to some degree.

List of features:

  • Powering on/off
  • Lock screen
  • Current time
  • Wi-Fi connections
  • Display settings (custom wallpapers, power off hotkey)
  • Security settings (find my device, ShareAllMyData™)
  • Accessibility settings (launch on Windows startup, "blind mode", "captions")
  • System settings ("language", "input methods", "measurement units")
  • About screen (credits, GitHub link, license)
  • "For developers"
  • Calculator
  • Browser
  • Phone dialing app (calls via 3rd-party programs, e.g. Skype)
  • Peartify the music app (local media; Spotify (buggy), SoundCloud)
  • Translator
  • Chattenger, allows you to open common social media websites
  • Pear.Maps, OpenStreetMaps integration (buggy)
  • yaWeather the weather app

Strikethrough indicates that a feature no longer works properly, mostly due to the absence of support and reliance on 3rd-party services. Also, some features are broken on Windows 7 (and probably 8); Windows Vista/XP and below are not supported at all.
